ANALISE FISCAL - versão Premium (tema verde analítico, landing C1, PDF P3)

Como usar:
1. Extraia esta pasta.
2. Entre na pasta: /analise_fiscal_site_premium
3. Crie e ative virtualenv:
   python -m venv venv
   venv\Scripts\activate  (Windows PowerShell)
4. Instale dependências:
   pip install -r requirements.txt
5. Rode:
   python app.py
6. Abra http://127.0.0.1:5000

Observações:
- O PDF premium usa ReportLab para compatibilidade em Windows.
